# New Session Handoff

When starting a new chat session for the film project, follow these steps:

## Step 1: Load Memory
// turbo
Read the memory README file:
```
View file: G:\0.cslab.ai\001.openClaw\film\memory\README.md
```

## Step 2: Load Core Files
// turbo
Read these files in order:
```
View file: G:\0.cslab.ai\001.openClaw\film\memory\SOUL.md
View file: G:\0.cslab.ai\001.openClaw\film\memory\CORRECTIONS.md
View file: G:\0.cslab.ai\001.openClaw\film\memory\DECISIONS.md
```

## Step 3: Load Recent Day Logs
// turbo
Check the days directory and read the most recent 2-3 day logs:
```
List directory: G:\0.cslab.ai\001.openClaw\film\memory\days\
```
Then read the most recent files found.

## Step 4: Load Topic Files (as needed)
// turbo
If the session will involve a specific topic, read the relevant topic file:
```
List directory: G:\0.cslab.ai\001.openClaw\film\memory\topics\
```
Available topics:
- studio-mechanism.md
- fourth-wall.md
- characters.md
- opening-scene.md
- ai-usage.md
- tone-and-feel.md

## Step 5: Confirm
Tell Su what you've read and what you understand. Don't summarize everything — just confirm you're up to speed and ask what he wants to work on today.

## Step 6: Before Ending Any Session
Before ending work, always:
1. Update or create today's day log in `film/memory/days/YYYY-MM-DD.md`
2. Update any topic files that changed
3. Add new corrections to `CORRECTIONS.md` if any mistakes were made
4. Add new decisions to `DECISIONS.md` if anything was finalized
5. Commit and push all memory file changes to git
